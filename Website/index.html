The Aberystwyth Web Evaluation Surveys Of Module Experiences

Introduction

The "Aberystwyth Web Evaluation Surveys Of Module Experiences", or AWESOME for short, is a web-based module evaluation questionnaire generator for the monitoring and evaluation of teaching.

At Aberystwyth University's Computer Science department, the evaluation of modules is done by the Staff Student Committee Convenor (Dr. Hannah Dee). The department has struggled with systems like Quizdom and paper forms in the past, but neither systems have satisfied the needs of the committee.

At the moment the department uses Google Forms in order to create forms in order for the students to complete. One issue with these forms is that we are currently asking students to ignore those questions which need-not apply to them, meaning that the student is forced to scroll endlessly until he/she finds their relevant module. The mundane process of seemingly endless scrolling has been pinpointed as to why the response rate to these questionnaires are so low (around 20% for each year).

We aim to collect honest data about student opinions, and to do this we will need to build a system that bases itself on anonymity and ease of use.


Features




Automatic student-specific questionnaire generation

Using a set of standard questions for each module, the system should generate student-specific questionnaires (each student may have different modules) as well as including additional departmental or institutional questions for good measure.


Generation of mid-term questionnaires

The system should be able to generate pre-SSCC questionnaires to determine which modules are going well. These mini-questionnaires will contain one question per module.


No registration details needed

Instead of having the student entering all of his/hers module information, AWESOME will automatically import module registration data via either AStRA (Aberystwyth Student Records and Admissions) or SAMS (Student Attendance Monitoring System).


Targeted follow-up reminder emails

People who've already completed the questionnaire shouldn't have to put up with endless reminder emails. One of the key features of the AWESOME system is that it can send email reminders to only those who are yet to respond.


Truly anonymous responses

Although we would like to know who is yet to respond as to send follow-up emails, we would not like to reconstruct what student had said what. Anonymity is vital in ensuring that we get honest data about what students really think.


Analytics

Reports should be made available to staff on a by-module, by-department and by-scheme basis, with automatically generated graphs and textual responses laid out nicely.
